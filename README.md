## Installation 
 
   1. fromthe root directory, npm i to add necessary dependancies
   2. Open the scema.sql file in new terminal, log into postgres, then \i schema.sql to initialize the database
   3. Return to the root folder, and ```node seeds/index.js``` to seed the seed files. 
   4. then npm start!

## Usage 


1. Clone the repo from github (link in the contact section) 
2. Run "node index.js" in the terminal
